# Homebrew Tap for TextLands

Install the TextLands CLI via Homebrew.

## Installation

```bash
brew tap MindFortressInc/textlands
brew install textlands
```

## Usage

```bash
textlands realms    # List available realms
textlands play      # Start playing
```
